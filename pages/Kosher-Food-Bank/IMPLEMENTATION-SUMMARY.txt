CRITICAL TECHNICAL NOTES:
--------------------------

1. CMS OVERRIDE FIGHTING:
   - The CMS continuously reapplies white colors to navigation links
   - Solution: setInterval running every 100ms to force black colors
   - Uses link.style.setProperty with 'important' flag
   - Removes -webkit-text-fill-color overrides

2. MOBILE MENU TOGGLE:
   - Button created with JavaScript and inserted into navigation
   - Shows "MENU" by default, changes to "CLOSE" when open
   - Toggles .menu-open class on .chabad_menu_content
   - Menu hidden by default on mobile (display: none)

3. PAGE DETECTION:
   - Kosher Food Bank: URL contains '6819985' or 'kosher-food-bank'
   - About Us: URL contains '6820886' or 'about-us'
   - Body class added for CSS scoping (prevents conflicts)

4. RESPONSIVE BREAKPOINTS:
   - Desktop: No media query (default)
   - Large tablet: max-width: 1024px
   - Tablet/Mobile: max-width: 768px
   - Small mobile: max-width: 480px

5. FONT FAMILY:
   - All text uses 'Urbanist', sans-serif
   - Loaded via Google Fonts in custom-header-code.html
